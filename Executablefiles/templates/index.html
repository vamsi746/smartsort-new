What is SmartSort?

An AI tool that detects fruit & vegetable freshness using deep learning and VGG16 model.

Python, Flask, TensorFlow, HTML/CSS, Gradio, VGG16, Render, GitHub

Use Cases

Helps farmers, grocers, vendors and households reduce waste and identify rotten produce.
